# TỰ ĐỘNG HÓA PHÂN TÍCH HỢP ĐỒNG VỚI DOCZY.AI TRÊN AWS

Doczy.ai™ là nền tảng phân tích tài liệu sử dụng Generative AI trên AWS để tự động xử lý hợp đồng. Giải pháp này giúp tăng đáng kể độ chính xác so với các hệ thống dựa trên luật truyền thống.

## Những điểm chính cần biết

- Amazon Cognito xác thực người dùng.
- Amazon S3 lưu trữ hợp đồng.
- AWS Lambda kích hoạt Amazon Textract.
- Amazon Textract trích xuất văn bản và metadata.
- Smart Chunking giữ nguyên cấu trúc tài liệu.
- Amazon ECS xử lý các tác vụ AI.
- Amazon Bedrock phân tích hợp đồng.
- Snowflake lưu trữ dữ liệu sau xử lý.
- Amazon CloudWatch giám sát hệ thống.
- AWS Secrets Manager bảo vệ thông tin nhạy cảm.

Kiến trúc này cho thấy cách các dịch vụ AI của AWS giúp tự động hóa quy trình phân tích hợp đồng trong doanh nghiệp.
