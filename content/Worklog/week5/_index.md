## Overview

This week focuses on:

* CloudWatch Metrics
* CloudWatch Logs
* CloudWatch Alarms
* CloudWatch Dashboard
* CloudWatch Advanced
* Amazon Managed Grafana
* EC2 Monitoring
* VPC Monitoring
* Monitoring RDS
* Monitoring S3
* Cost Monitoring

## Objectives

* Learn how Amazon CloudWatch monitors AWS resources.
* Configure metrics, logs, alarms, and dashboards.
* Monitor EC2, VPC, RDS, and S3 resources.
* Explore advanced monitoring with Amazon Managed Grafana.
* Understand AWS cost monitoring and service quotas.
* Build a centralized monitoring solution for AWS workloads.
