## Tôi đã làm

- Tìm hiểu cách cấu hình Security Group trong AWS Management Console.
- Xem cách tạo và chỉnh sửa Security Groups.
- Thực hành thêm các quy tắc Inbound và Outbound.
- Kiểm tra các trường hợp truy cập phổ biến cho dịch vụ Web và SSH.
- So sánh các cấu hình Security Group khác nhau.

## Kết quả

- Làm quen với việc cấu hình Security Groups.
- Cải thiện hiểu biết về kiểm soát truy cập mạng.
- Hiểu cách Security Groups được áp dụng cho EC2 Instance.
- Tự tin hơn trong việc cấu hình các quy tắc tường lửa cơ bản.
